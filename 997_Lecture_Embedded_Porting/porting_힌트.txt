	기존 커널에서 복사

6. low-level debuging
	make menuconfig
	Kernel hacking  -->
		[*] Kernel low-level debugging functions (read help!)
			Kernel low-level debugging port (Use Samsung S3C UART 1 for low-level debug)  --->

	## printch 사용 예 (arch/arm/kernel/head.S 같은 어셈블리어에서 사용 가능)
		mov  r0, #65
		bl printch

7. early_printk
	make menuconfig
	Kernel hacking  --->
		[*] Early printk

	## early_printk() 함수를 사용하기 위해서는 U-Boot에서 bootargs에 "earlyprintk" 를 추가해주어야 함
	
	## early_printk() 사용 예 (init/main.c에서 사용 가능, 단 setup_arch(&command_line); 이후에 가능)
		early_printk("[%s:%d]-%s()\n", __FILE__,__LINE__,__FUNCTION__);
